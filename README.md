A Node.js + Express backend for a real‑time chat application.  
This service handles authentication, REST endpoints, WebSocket communication, and database persistence.

---

## Features

- User registration and login (JWT-based)
- Socket.io real‑time messaging
- Room‑based chat system
- Online user tracking
- Message persistence using PostgreSQL
- Sequelize ORM models and migrations
- Clean separation of routes, controllers, and services

---

## Tech Stack

- **Node.js**
- **Express**
- **Socket.io**
- **PostgreSQL**
- **Sequelize**
- **JWT Authentication**
- **bcrypt** for password hashing
